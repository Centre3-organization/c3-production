# SAP Fiori Design Patterns for Forms

## Key Design Principles
1. **Object Page Layout** - Header with object info, tabbed sections below
2. **Form Groups** - Logical grouping with section headers (Basic Data, Communication Data, etc.)
3. **Label-above-input** - Labels positioned above input fields with colon suffix
4. **Clean white background** - Light/white form backgrounds with subtle borders
5. **Horizontal sections** - Multiple columns (2-3) for related fields
6. **Section headers** - Underlined section titles to separate form groups
7. **Tables for related data** - Use tables for lists (Bank Accounts, Contacts, etc.)

## Input Field Patterns
- Labels above inputs with colon (e.g., "Document Currency:")
- Placeholder text in italics (e.g., "Enter currency code")
- Subtle border (1px light gray)
- Dropdown indicator (chevron) on right for selects
- Value help icon for lookup fields

## Button Patterns
- Action buttons: outlined style
- Emphasize buttons: filled primary color
- Menu buttons with dropdown
- Toggle/switch controls

## Colour Palette (SAP Fiori)
- Primary blue: #0a6ed1
- Dark navy: #354a5f
- Light background: #f7f7f7
- White: #ffffff
- Border: #e5e5e5

## For Centre3 Branding
- Keep Centre3 primary color (pink/magenta gradient)
- Use Poppins font
- Apply Fiori layout patterns with Centre3 colors
