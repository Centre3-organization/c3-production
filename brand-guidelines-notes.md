# Center3 Brand Guidelines - Key Specifications

## Logo
- Brand name: "center3" (lowercase)
- Logo consists of a diamond/arrow icon with coral center + "center3" text
- Positive version: Purple logo with coral accent on white background
- Negative version: White logo with coral accent on purple background
- Monochromatic versions available (black on white, white on black)

## Typography
- **Font Family:** Latin type family (appears to be a sans-serif similar to Montserrat)
- **Weights available:** Thin, Light, Regular, Medium, Bold, Extrabold
- **Usage guidelines:**
  - Thin: Legal texts or large body copies
  - Light, Regular, Medium: Standard communication, headlines, body copies
  - Bold, Extrabold: Highlights, numbers, percentages, one-word headlines
- Avoid using thick weights (bold & extrabold) for headlines to keep message clearer

## Notes
- Logo uses a distinctive diamond shape with an arrow pointing right
- The coral accent is in the center of the diamond icon
- Brand name is always written as lowercase "center3"
- Coral dot must always contrast with the background

## Iconography
- Main design is outline style
- Use filled design to show different state (e.g., active or hover)
- Icon sizes: 16px to 32px
- Icons should be in purple color (#4f008c) for default state
- Icons use simple, clean line style
- Responsive icons adapt between outline and filled states

## Brand Application Summary
1. **Primary Purple (#4f008c)** - Main brand color for headers, buttons, icons
2. **Coral (#ff375e)** - Accent color for highlights, CTAs, important elements
3. **White (#ffffff)** - Primary background
4. **Silver (#8e9aa5)** - Secondary text, borders
5. **Light Silver (#dedfe2)** - Subtle backgrounds, dividers
6. **Dark Purple (#3a1066)** - Dark mode backgrounds, illustrations
7. **Light Purple (#7333a3)** - Supergraphic elements, gradients
8. **Moon Light (#a54ee1)** - Secondary purple for accents
