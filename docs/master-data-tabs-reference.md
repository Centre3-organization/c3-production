# Master Data Tabs Reference (from screenshot)

## Tab Order (left to right):
1. **Activities** - with sub-tabs: Main Activities, Sub-Activities
2. **Role Types**
3. **Approvers**
4. **Countries**
5. **Regions**
6. **Cities**
7. **Site Types**
8. **Zone Types**
9. **Area Types**

## Activities Tab Features:
- Main Activities sub-tab (default)
- Sub-Activities sub-tab
- "Add Main Activity" button (purple)
- Table columns: Main Activity Name, Description, Sub-Activities (count), Status, Actions
- Each row has icon, edit and delete buttons
- Status badges: Active (green), Inactive (purple/gray)

## Sample Main Activities from screenshot:
- Cabling Changes - Activities related to cable installation, removal, and modifications
- Checks, Visit & Audit - Inspection, audit, and visit activities
- Facilities & Civil Work - Building and infrastructure related activities
- HW Changes - Hardware installation, removal, and modifications
- Input / Output - Material entry and exit activities
- Security & Safety Changes - Security and safety related activities
- IT System & Software - IT system and software related activities
